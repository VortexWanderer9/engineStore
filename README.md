## About this small project 
This is e-commerce website built using ReactJS for frontend and NodeJS for backend. The project is intended to demonstrate a simple e-commerce application with basic functionalities such as product listing, user authentication, and shopping cart management.

## Features
- User Registration and Login
- Product Listing and Search
- Shopping Cart Management
- Order Placement
- Responsive Design
## Technologies Used
- Frontend: ReactJS, Redux, Bootstrap
- Backend: NodeJS, ExpressJS
- Database: MongoDB
